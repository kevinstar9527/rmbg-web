# 背景去除工具 (Background Removal Tool)

一个简单易用的网页应用，可以自动去除图片背景。使用 开源AI模型，实现高质量图像分割。

## 特点

- 使用 BRIA RMBG-1.4 ONNX 模型进行高质量图像分割
- 集成式服务设计，一键启动
- 支持拖放、文件选择和剪贴板粘贴等多种图片输入方式
- 提供对比滑块，方便查看处理前后效果
- 自适应布局，支持移动端和桌面端
- 一键下载处理后的透明背景图片

## 使用方法


1. 运行 `npm install` 安装依赖。
2. 运行 `npm start` 启动应用。

3. 通过以下任意方式上传图片:
   - 点击"选择文件"按钮选择图片
   - 将图片拖放到指定区域
   - 点击"从剪贴板粘贴"或直接按 Ctrl+V 粘贴剪贴板中的图片

4. 等待处理完成
5. 使用对比滑块查看处理效果
6. 点击"下载图片"保存处理结果


## 系统要求

- 支持 ONNX Runtime 的系统
- 现代浏览器 (Chrome, Firefox, Edge, Safari 等)

## 模型信息

该应用使用 BRIA RMBG-1.4 模型进行图像分割。

- `model_quantized.onnx`: 量化后的模型文件
